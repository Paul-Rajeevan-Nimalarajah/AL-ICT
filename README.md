# AL ICT Notes Hub

**AL ICT Notes Hub** is a comprehensive educational resource platform dedicated to G.C.E. Advanced Level ICT students in the Tamil Medium.

### 👤 Developed & Maintained by
**Paul Rajeevan** - [paulrajeevan.com](https://paulrajeevan.com)

---

## 🚀 Features

- **Unit Notes**: Verified, syllabus-wise notes for Units 1 through 14.
- **Past Papers Archive**: A collection of official A/L ICT past papers (Part I & II) from 2011 to 2024 with marking schemes.
- **Model & Term Papers**: Practice papers from various provinces and schools to help with exam preparation.
- **Tuition Connect**: A free directory for physical and online ICT tuition classes across Sri Lanka.
- **Contribution Hub**: A platform for students and teachers to submit their own notes and resources.

## 📂 Project Structure

- `/website`: The core web application built with HTML, CSS, and Vanilla JS.
  - `index.html`: Home page and main navigation.
  - `notes.html`, `past-papers.html`, `model-papers.html`: Resource dedicated pages.
  - `tuition.html`: Tuition advertisement directory.
  - `contact.html`: Dynamic contact page with support for GitHub/LinkedIn links and Google Forms integration.
- `/android-app`: An Android application (WebView wrapper) for accessing the hub on mobile devices.

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+), FontAwesome Icons.
- **SEO**: Fully optimized with meta tags, Open Graph (OG) support, JSON-LD structured data, robots.txt, and XML Sitemap.
- **Deployment**: Configured for GitHub Pages with automated CI/CD via GitHub Actions.

## 🧪 Local Development

To run the website locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/Paul-Rajeevan-Nimalarajah/AL-ICT.git
   ```
2. Navigate to the `website` directory:
   ```bash
   cd website
   ```
3. Use a local server (e.g., Live Server or `npx serve`):
   ```bash
   npx serve
   ```
