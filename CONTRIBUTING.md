# Contributing to AL ICT Notes Hub

First off, thank you for considering contributing to the AL ICT Notes Hub! It's people like you that make this platform a great resource for the A/L ICT Tamil Medium community. We welcome contributions from students, teachers, and developers to help improve this site!

## How Can I Contribute?

### 1. Submit Educational Notes & Resources
If you are a student or a teacher with quality notes, past papers, or model papers, you can submit them to be hosted on the hub.
- **How to submit**: You can use the [Contact Form](https://alict.paulrajeevan.com/contact.html) on our website to submit your materials.
- **Credit**: You will receive full credit by name on every resource you submit.

### 2. Add or Update Tuition Listings
If you know of a good ICT tuition class (physical or online), let us know so we can list it for free in our Directory.
- **How to list**: Use the [Tuition Registration Form](https://alict.paulrajeevan.com/contact.html).

### 3. Report Bugs or Issues
Found a broken link, a typo, or a bug on the website?
- **How to report**: Open an [Issue](https://github.com/Paul-Rajeevan-Nimalarajah/AL-ICT/issues) on this GitHub repository, or use the Contact Form on the website.

### 4. Code Improvements (For Developers)
If you want to improve the website's design, add new features, or optimize performance, we highly encourage Pull Requests!

#### Local Development Setup
1. Fork the repository and clone your fork:
   ```bash
   git clone https://github.com/YOUR-USERNAME/AL-ICT.git
   ```
2. Navigate to the `website` directory:
   ```bash
   cd website
   ```
3. Run a local server to view your changes:
   ```bash
   npx serve
   ```
4. Create a new branch for your feature: `git checkout -b feature/your-feature-name`
5. Make your code changes.
6. Commit your changes: `git commit -m 'Add your feature description'`
7. Push to the branch: `git push origin feature/your-feature-name`
8. Open a Pull Request on the main repository.
